About the System

Comprehensive oral cancer prediction using advanced machine learning

System Overview

This web application provides a comprehensive set of machine learning models for oral cancer prediction, treatment cost estimation, survival rate prediction, and recovery time forecasting.

Available Models

Cancer Label Detection

Binary classification model using Logistic Regression to detect the presence of oral cancer based on patient features and symptoms.

Treatment Cost Prediction

Random Forest Regressor model that estimates treatment costs in USD based on patient characteristics and treatment type.

Survival Rate Prediction

Deep Neural Network model that predicts 5-year survival rates as a percentage.

Length of Stay Prediction

Regression model that forecasts the duration of hospital stay in days.

Recovery Time Prediction

Regression model that predicts the recovery time in days post-treatment.

Features

  • Multiple machine learning models in one integrated platform
  • Real-time predictions with instant results
  • Model health monitoring and status tracking
  • Automatic model validation and sanity testing
  • Professional and intuitive user interface
  • Robust error handling and logging

Technology Stack

Backend: Flask (Python web framework)

Machine Learning: scikit-learn, TensorFlow/Keras, XGBoost

Data Processing: Pandas, NumPy

Model Storage: Joblib/Pickle

Frontend: HTML5, CSS3, JavaScript
